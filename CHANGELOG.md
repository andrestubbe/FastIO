# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-02
### Added
- **Blueprint Alignment** — Full repository reorganization to match FastJava standards.
- **FastCore Integration** — Unified native library loading.
- **Native DLL Builder** — Added `compile.bat` and `native/fastio.def`.
- **Examples** — Moved Demo and Benchmark to dedicated `examples/` folder.
- **Package Flattening** — Migrated from `io.github.andrestubbe.fastio` to `fastio`.

## [0.1.0] - 2026-04-13
### Added
- Initial release with native Windows I/O acceleration.
- Unbuffered I/O support.
- Memory-mapped files.
- Direct ByteBuffer operations.
